# xstate-preact-signals

Iterating over a few ideas for using

[XState](https://xstate.js.org/docs/) with [Preact Signals](https://preactjs.com/guide/v10/signals/)

## Install

```sh
pnpm install
```

## Run

```sh
pnpm dev
```

Available on http://127.0.0.1:3001/two-state

## Also using

- Preact
- Vite
- Playwright Test
- TypeScript
- Tailwindcss
- pnpm
